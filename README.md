# Musical Moods App

## URL to live site: 

https://wondrous-semolina-cdca20.netlify.app/

## Description

The Musical Moods App allows users to search for a song by title and artist. The app will return the title of the album the song was on and an image of the album cover. It will also return an image that represents the "mood" of the song, based on the valence measure provided by the Spotify API map.


## Technologies Used

The backend of this application was built with Python Flask. The [frontend](https://github.com/kb789/Book-Cloud-Frontend) was built with React. Firebase was used to manage authentication, and a Firebase database was used for storage.


## APIs used

The WorldCat Search API was used to get the title, author, subject, and description information for the books.
